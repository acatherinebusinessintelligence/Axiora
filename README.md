# Axiora GitHub Package

Este paquete está listo para subir a un repositorio y publicar con GitHub Pages.

## Contenido
- `index.html`: página principal
- `assets/`: imágenes usadas por la página

## Cómo publicarlo en GitHub Pages
1. Crea un repositorio nuevo en GitHub.
2. Sube todo el contenido de esta carpeta.
3. Verifica que el archivo principal se llame `index.html`.
4. En GitHub entra a **Settings > Pages**.
5. En **Build and deployment**, selecciona:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / `(root)`
6. Guarda los cambios.
7. GitHub te entregará la URL pública del sitio.

## Nota
La página ya quedó configurada para cargar las imágenes desde la carpeta `assets/`.
